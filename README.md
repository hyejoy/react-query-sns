# 🐾 Hanip Log (한입 로그)

A mini SNS project built with React.js + TypeScript + Supabase, focused on mastering TanStack Query for server state management.

> 인프런 "한 입 크기로 잘라먹는 React.js 실전 프로젝트 - SNS 편" 강의 실습 프로젝트 (섹션 5 ~ 12)

---

## 🛠 Tech Stack

- **React.js** + **TypeScript**
- **Supabase** — 인증, DB, 스토리지
- **TanStack Query** — 서버 상태 관리
- **Zustand** — 전역 상태 관리
- **Tailwind CSS** + **Shadcn/ui** — 스타일링
- **React Router V7** — 라우팅
- **Vite** — 빌드 도구
